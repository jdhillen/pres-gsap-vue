What is GSAP?

The GreenSock Animation Platform, or GSAP, is a powerful Javascript animation library that helps developers build out engaging and performant animations.

What can GSAP Animate?
  • CSS Properties
  • SVG
  • canvas
  • generic objects
  • Whatever!
Why GSAP over other animation libraries?
  • Platform agnostic
  • 63kB minified bundle size
  • 20 times faster than jQuery
  • Animation Library Speed Tests
Requirements
  • Node.js 10x or higher
  • yarn or npm
  • Basic Javascript Knowledge
Installation
            
              yarn add gsap
            
          
             npm install gsap 
          
             import { gsap } from "gsap";

Timelines
            
              // WITHOUT Timelines (only using tweens with delays):
              gsap.to("#id", {x: 100, duration: 1});
              gsap.to("#id", {y: 50, duration: 1, delay: 1});
              gsap.to("#id", {opacity: 0, duration: 1, delay: 2});
vs 
            
              //WITH Timelines (cleaner, more versatile)
              var tl = gsap.timeline({repeat: 2, repeatDelay: 1});
              tl.to("#id", {x: 100, duration: 1}, "-=0.5");
              tl.to("#id", {y: 50, duration: 1}), "-=0.5";
              tl.to("#id", {opacity: 0, duration: 1}, "-=0.5");
